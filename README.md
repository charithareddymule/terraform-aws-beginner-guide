This project demonstrates the basics of Infrastructure as Code (IaC) using Terraform and AWS.

The infrastructure was created in an AWS Sandbox environment using AWS Cloud9.

---

## What I did
- Created AWS Cloud9 environment
- Configured AWS credentials using environment variables
- Wrote Terraform configuration to create an EC2 instance
- Executed Terraform workflow: init, plan, apply, destroy
- Verified resource creation in AWS EC2 Console

---

## Tech Stack
- Terraform
- AWS (EC2)
- AWS Cloud9
- HTML & CSS
